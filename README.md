# Practical AI Skills

Static GitHub Pages site and matching one-sided business card for a QR-first AI skills promotion.

## Files

- `index.html` - public website
- `styles.css` - website styling
- `business-card.html` - printable 3.5 x 2 inch business-card side
- `card.css` - business-card styling
- `assets/ai-skills-hero.png` - generated website/card visual
- `assets/qr.svg` - QR code for the published site

## Current QR Target

`https://batchscribe-dev.github.io/ai-skills/`

Update the QR target before publishing if the GitHub username or repo slug should be different.
